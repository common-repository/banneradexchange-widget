== Description ==

A widget that enables you to display banner code from BannerAdExchange.
You need to put your publisher ID, and shoose banner size.

== Installation ==

Installation is easy:

* Download the plugin or search for "BannerAdExchange" in your WordPress admin panel
* If you downloaded it, copy the folder 'BannerAdExchange' to the plugins directory of your blog
* Activate the plugin through the 'Plugins' menu in WordPress
* In the "Widgets" screen, drag the "BannerAdExchange" to the widget area of your choice
* Configure the widget from the widget panel as required
